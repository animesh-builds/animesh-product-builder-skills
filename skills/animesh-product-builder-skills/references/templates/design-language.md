# [Project Name] — Design Language

## Foundations

### Color
Primary, secondary, neutrals (50–900), semantic (success/warning/error/info), light + dark tokens.

### Typography
Font families, type scale, weight usage, line-height.

### Spacing
Base unit (4px or 8px) and scale.

### Radius / Elevation
Border radius and shadow tokens.

### Motion
Default easing and duration. Usage policy.

## Components
Conventions for: buttons, inputs, cards, modals, navigation, empty states, loading states.
For each: default / hover / active / disabled / focus-visible.

## Iconography
Library, stroke width, size scale.

## Breakpoints
Mobile / tablet / desktop cutoffs.

## Accessibility
- WCAG target (typically 2.1 AA)
- Contrast targets
- Focus rings
- Minimum tap target (44×44)
- Keyboard nav expectations
- Screen reader expectations for custom components
